# BGDev Official Website

Modern, responsive and animated website for BGDev - AI-Powered Solutions company.

## 🚀 Features

- **Modern Design**: Dark theme with glassmorphism effects
- **Advanced Animations**: Framer Motion powered smooth animations
- **Responsive**: Fully responsive design for all devices
- **AI-Powered Theme**: Circuit board lightbulb logo and tech-focused design
- **Performance Optimized**: Built with React + TypeScript + Tailwind CSS

## 🛠️ Technologies Used

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Create React App

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/bgasimzade99/bgdevofficial.git
cd bgdevofficial/bgdev-react
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## 🎨 Design Features

- **Hero Section**: Animated geometric shapes and floating particles
- **Services**: Interactive cards with hover effects
- **Projects**: Portfolio showcase with filtering
- **Contact**: Modern contact form with animations
- **Footer**: Comprehensive links and social media integration

## 📱 Responsive Design

The website is fully responsive and optimized for:
- Desktop (1200px+)
- Tablet (768px - 1199px)
- Mobile (320px - 767px)

## 🚀 Deployment

Build the project for production:
```bash
npm run build
```

The build folder contains the production-ready files that can be deployed to any static hosting service.
